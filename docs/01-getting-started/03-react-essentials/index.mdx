To build applications with Next.js, it helps to be familiar with React and some of its newer features.

If you're new to React, we recommend going through our [Next.js Foundations Course](/learn/foundations/from-javascript-to-react) which covers fundamental concepts like JSX, Components, Props, State, and Hooks.

We also recommend these great resources for learning React:

- [React Tutorial](https://react.dev/learn/tutorial-tic-tac-toe)
- [Thinking in React](https://react.dev/learn/thinking-in-react)
- [Learn React](https://react.dev/learn/describing-the-ui)
